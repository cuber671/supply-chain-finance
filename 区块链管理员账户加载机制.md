# FISCO BCOS 区块链管理员账户加载机制

## 1. 概述

本文档详细说明系统中区块链管理员账户的配置、加载机制及代码复用方式。所有区块链写操作都需要使用管理员密钥对进行签名，因此理解此机制对于开发和扩展新功能至关重要。

## 2. 账户配置

### 2.1 配置文件位置

主配置文件位于 `docker/config/config.toml`：

```toml
[account]
    # 密钥文件目录（相对于SDK配置目录）
    keyStoreDir = "account"
    # 管理员账户地址
    accountAddress = "0x6976c14175ddfae575ffab3845d89e155e09569d"
    # 密钥文件格式
    accountFileFormat = "pem"
```

### 2.2 密钥文件位置

- **Docker容器内**: `/app/sdk/account/0x6976c14175ddfae575ffab3845d89e155e09569d.pem`
- **宿主机映射**: `src/main/resources/account/0x6976c14175ddfae575ffab3845d89e155e09569d.pem`

### 2.3 相关配置项

| 配置项         | 值            | 说明                    |
| -------------- | ------------- | ----------------------- |
| cryptoType     | 0             | 0为标准ECDSA，1为国密SM |
| keyStoreDir    | account       | 密钥文件目录            |
| accountAddress | 0x6976c141... | 管理员账户地址          |

## 3. 加载流程

### 3.1 优先级顺序

```
1. 从 Client 对象反射获取密钥对
         ↓
2. 从 SDK 对象反射获取密钥对
         ↓
3. 从配置文件读取PEM文件加载
         ↓
4. 生成随机密钥对（仅查询可用）
```

### 3.2 启动日志示例


## 4. 核心代码实现

### 4.1 BlockchainConfig 配置类

**文件**: `src/main/java/com/fisco/app/Common/Config/BlockchainConfig.java`

```java
@Bean
public CryptoKeyPair cryptoKeyPair(CryptoSuite cryptoSuite, BcosSDK bcosSDK, Client client) {
    // 方案1: 尝试从 Client 获取
    if (client != null) {
        CryptoKeyPair keyPair = tryGetKeyPairFromClient(client);
        if (keyPair != null) {
            logger.info("从 Client 获取密钥对成功，地址: {}", keyPair.getAddress());
            return keyPair;
        }
    }

    // 方案2: 从配置文件读取PEM文件
    CryptoKeyPair keyPair = loadKeyPairFromConfig(cryptoSuite);
    if (keyPair != null) {
        logger.info("从配置文件加载密钥对成功，地址: {}", keyPair.getAddress());
        return keyPair;
    }

    // 方案3: 生成随机密钥对
    return cryptoSuite.getCryptoKeyPair();
}
```

### 4.2 BaseContractService 基类

**文件**: `src/main/java/com/fisco/app/Common/Service/BaseContractService.java`

```java
public abstract class BaseContractService {

    @Autowired
    protected Client client;

    @Autowired
    protected CryptoKeyPair cryptoKeyPair;

    /**
     * 获取当前账户地址

    protected String getCurrentAccountAddress() {
        return cryptoKeyPair != null ? cryptoKeyPair.getAddress() : null;

     */
    protected TransactionResponse sendTransactionWithAudit(
            Contract contract,
            String functionName,
            Object[] params,
            String operationType) {

        // ...
    }
}
```

### 4.3 EnterpriseContractService 示例

**文件**: `src/main/java/com/fisco/app/Modules/Enterprise/Service/EnterpriseContractService.java`

```java
@Service
public class EnterpriseContractService extends BaseContractService {

    @Value("${contract.enterprise}")
    private String enterpriseContractAddress;

    private EnterpriseRegistryV2 enterpriseContract;

    @javax.annotation.PostConstruct
    public void init() {
        if (client == null || cryptoKeyPair == null) {
            logger.error("区块链客户端未初始化，无法加载企业合约");
            return;
        }

        logger.info("使用 SDK 密钥对，地址: {}", cryptoKeyPair.getAddress());

        // 加载合约，使用 cryptoKeyPair 进行交易签名
        this.enterpriseContract = EnterpriseRegistryV2.load(
                enterpriseContractAddress,
                client,
                cryptoKeyPair  // 关键：管理员密钥对
        );

        logger.info("企业合约加载成功，地址: {}", enterpriseContractAddress);
    }

    /**
     * 注册企业上链
     */
    public TransactionReceipt registerEnterprise(...) {
        TransactionResponse response = sendTransactionWithAudit(
                enterpriseContract,
                "registerEnterprise",
                new Object[]{input},
                "ENTERPRISE_REGISTER"
        );
        return response != null ? response.getTransactionReceipt() : null;
    }
}
```

## 5. 开发新模块指南

### 5.1 方式A：继承 BaseContractService（推荐）

```java
@Service
public class NewContractService extends BaseContractService {

    @Value("${contract.new}")
    private String contractAddress;

    private NewContract contract;

    @PostConstruct
    public void init() {
        // 使用继承的 cryptoKeyPair 加载合约
        this.contract = NewContract.load(contractAddress, client, cryptoKeyPair);
    }

    /**
     * 执行链上操作
     */
    public TransactionReceipt doSomething(String param) {
        return sendTransactionWithAudit(
            contract,
            "methodName",
            new Object[]{param},
            "OP_TYPE"
        );
    }
}
```

### 5.2 方式B：直接使用 BlockchainConfig

```java
@Autowired
private BlockchainConfig blockchainConfig;

/**
 * 生成新密钥对（用于企业注册时创建区块链身份）
 */
public CryptoKeyPair generateKeyPair() {
    return blockchainConfig.generateKeyPair();
}
```
